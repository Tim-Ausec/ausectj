But first, a prelude about the philosophy behind these choices

Minimalism

For those who haven’t read the “Why I Made This Website” article, there are three main values that led me to create this website:

  1. The value of freedom in the digital world
  2. The value of learning
  3. The value of self-expression

But there is a fourth value that has led me towards minimalism as the primary design philosophy for this website: 4. The value of purpose/focus

Minimalism is the embodiment of this value in design. The primary goal of minimalism is focus, and cutting out all distractions except the core is its method.

The core of this website is my self expression - my ideas and interests. Therefore, I won’t bother with ideal formatting, user-friendliness, or dynamic websites - all of these distract from the purpose.

As the cherry on the top, minimal, static websites load quicker, are cheaper, and come with less headache :)

End prelude!

Domain and Hosting

This is simple - I just went on Reddit and found a guide that convinced me. My main concern was price and minimizing serious headaches.

Another perk of static websites is that DigitalOcean allows you to host up to 3 for free. Forever(?). So I just have to pay for the domain name!

Porkbun is cheap and seems reputable, so I’m happy there as well.

The only thing I’ve been unhappy with so far is that it took like 2 weeks for digital ocean to propagate as an authorized name server on my domain. I’m not sure if this is a common issue or is just because I’m bumming off of the free hosting, but it worked out so I’m not too worried about it.
